Frontend

É onde a mágica acontece!

É a arte de criar experiências visuais que encantam os usuários. Com HTML, CSS e JavaScript, os desenvolvedores transformam ideias em interfaces incríveis.

Essa combinação de tecnologias torna a web um lugar mais dinâmico e envolvente, surpreendendo os usuários a cada clique!

Ferramentas Web

HTML

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas da web. Ele estrutura o conteúdo. É a base sobre a qual todo o conteúdo da web é construído.

CSS

CSS (Cascading Style Sheets) é a linguagem usada para estilizar páginas da web. Com CSS, é possível definir cores, fontes, espaçamentos e layouts, tornando as páginas mais atraentes visualmente. Ele separa a apresentação do conteúdo, permitindo uma organização mais eficiente.

JavaScript

JavaScript é uma linguagem de programação que permite adicionar interatividade às páginas da web. Com ele, é possível criar animações, validar formulários e responder a eventos do usuário. É essencial para construir experiências dinâmicas e envolventes na web.

Frameworks

React

React é uma biblioteca JavaScript para criar interfaces de usuário através de componentes reutilizáveis.

React

Angular

Angular é um framework para aplicações web desenvolvido pelo Google, que permite criar aplicações robustas e escaláveis utilizando TypeScript.

Angular

Vue.js

Vue.js é um framework progressivo para construção de interfaces de usuário, que facilita a integração com outras bibliotecas e projetos existentes.

Vue.js

Bootstrap

Bootstrap é um framework front-end para desenvolvimento de sites responsivos, que inclui uma coleção de ferramentas para CSS e JavaScript.
